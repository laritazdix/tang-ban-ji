# 糖伴记：家庭血糖与胰岛素记录 MVP

这是一个给糖尿病家庭自我管理使用的轻量原型，重点是降低记录阻力：

- 快速记录血糖、胰岛素剂量、注射部位、影响因素和备注。
- 自动按医生目标标记“目标内 / 偏高 / 明显高血糖 / 低血糖”。
- 生成最近 14 天趋势图和近 7 天复诊摘要。
- 支持浏览器本地提醒、CSV 导出、打印/PDF。
- 数据只保存在当前浏览器 `localStorage`，不会上传到网络。

## 使用方法

直接用浏览器打开 `index.html` 即可试用。

如果希望导出、通知和 PWA 行为更稳定，可以在本目录启动一个本地静态服务器：

```powershell
python -m http.server 8080
```

然后访问：

```text
http://localhost:8080
```

## 医疗安全边界

本工具只做记录、提醒和整理，不自动推荐或调整胰岛素剂量。胰岛素调整应遵循医生给出的个体化方案。若出现意识障碍、无法进食、持续低血糖、酮症风险或血糖反复显著异常，请按医生方案处理并及时就医。

## 后续迁移到微信小程序

仓库里已经附带 `wechat-miniprogram/`，可以在微信开发者工具中以小程序项目导入。它是一个单页小程序 MVP，覆盖快速记录、目标提醒、今日时间线和复诊周报。

网页原型与微信小程序版使用同一类数据结构：

- `entries`：血糖和胰岛素记录集合。
- `reminders`：每日提醒集合。
- `oneShotReminders`：一次性提醒集合。
- `settings`：患者称呼、目标范围、医生规则。

微信小程序版建议使用：

- 页面：`首页记录`、`趋势`、`周报`、`设置`。
- 存储：微信云开发数据库或家庭私有后端。
- 提醒：微信订阅消息，替代当前浏览器打开时才可靠触发的提醒。
- 安全：所有剂量调整规则只展示医生录入内容，不由程序自动推断。
